# LifePilot AI

## 📋 Descrição
O LifePilot AI é um assistente pessoal inteligente desenvolvido para auxiliar usuários na organização de tarefas, compromissos, estudos e produtividade, utilizando Inteligência Artificial.

---

## 🎯 Objetivos

- Gerenciar tarefas
- Organizar compromissos
- Planejar estudos
- Gerar sugestões inteligentes
- Criar resumos diários
- Auxiliar na produtividade

---

## 🛠️ Tecnologias

### Front-end
- React
- Vite

### Back-end
- Node.js
- Express

### Banco de Dados
- PostgreSQL
- Prisma ORM

### Inteligência Artificial
- OpenAI API

### Autenticação
- JWT
